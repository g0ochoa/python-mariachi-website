Since Asana Free doesn't support custom fields, we use structured task naming:

### **Task Naming Format**
```
[Epic#] [Points] Task Name
```

**What the codes mean:**
- **[E1]** = Epic 1 (Technology Stack & Architecture)
- **[8]** = 8 Story Points (effort estimate using Fibonacci: 1,2,3,5,8,13,21)

### **Epic Reference**
- **E1**: Technology Stack & Architecture
- **E2**: Development Environment Setup  
- **E3**: Project Foundation & Infrastructure
- **E4**: Authentication & Security Framework
- **E5**: Public Website Development
- **E6**: Musicians Portal Development
- **E7**: Advanced Features & Integration
- **E8**: Infrastructure as Code & Deployment

### **Story Points Scale**
- **[1-2]**: Simple tasks (1-4 hours)
- **[3-5]**: Medium tasks (1-2 days)
- **[8-13]**: Complex tasks (3-5 days)
- **[21+]**: Break down into smaller tasks

## **Step 4: Task Assignment Tips**

**Important**: Asana creates separate tasks when you assign to multiple people. This is normal behavior.

- **Individual tasks**: Assign to one person per task
- **Team collaboration**: Use comments and @mentions
- **Progress tracking**: Use subtasks for breaking down work
- **Dependencies**: Link tasks that depend on each other
